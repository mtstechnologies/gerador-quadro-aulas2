# **App Name**: SchoolTime Optimizer

## Core Features:

- Professor Management: Allow admin to manage professor information with their basic details, subjects, and availability, considering multiple subjects and preferences for shifts.
- Subjects Management: Administer the registry of subjects and workload, defining workload, codes, names and if the subject requires any special room.
- Classes Management: Classes administration with features for registry of class, shift, number of students; assign the required subjects, define fixed schedules, and validate minimun and maximun capacity
- Automatic Schedule Generation: Automatically generate optimized schedules taking in consideration all the configured restrictions. Generates multiple schedule options.
- Restriction Configuration Tool: Allow for defining institutional rules and restrictions, set up consecutive classes, schedule preferences, blocked times and weights.
- Schedule Visualizer: Schedule Visualization multi-format allowing view by professor, student and class with intuitive schedule grids and PDF exporting.
- Schedule Adjustment and Conflict Resolution: Manual Schedule adjustments in a drag-and-drop interface, including validation of conflicts and change history.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5), conveys trust, authority, and intelligence, creating a professional yet approachable feel.
- Background color: Light gray (#F5F5F5) provides a neutral and clean backdrop.
- Accent color: Orange (#FF9800), use sparingly for interactive elements.
- Headline font: 'Space Grotesk' (sans-serif) will give the page a modern and computer-science related theme. Use 'Inter' for body.
- Use consistent icons from 'lucide-react' to represent entities and actions.
- Modern and structured layouts, ensuring that all elements are clearly visible.
- Subtle transitions to create visual appeal
